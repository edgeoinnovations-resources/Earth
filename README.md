# Earth

A 3D interactive globe built with CesiumJS, featuring satellite imagery basemap and world terrain.

## Features

- 3D globe with satellite imagery
- World terrain with elevation data
- Interactive navigation (zoom, pan, rotate)
- Base layer picker for different imagery options
- Geocoder for location search
- Scene mode picker (3D, 2D, Columbus View)
- Sun-based lighting

## View Live

Visit: https://edgeoinnovations-resources.github.io/Earth/

## Technology

Built with [CesiumJS](https://cesium.com/cesiumjs/) - an open-source JavaScript library for 3D globes and maps.
